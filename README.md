# SCD30

Driver for the Sensirion SCD30 sensor module, for monitoring CO2, temperature, and humidity.

## Usage

Installation: `toit.pkg install github.com/qvisten999/scd30`

```
import scd30

main:
  ...
```

See the `examples` folder for examples.
